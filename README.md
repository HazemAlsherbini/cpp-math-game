# Math Game

A console-based Math Quiz game developed in C++

## Features

- Generate random math questions.
- Multiple difficulty levels:
  - Easy.
  - Medium.
  - Hard.
  - Mix.
- Multiple operation types:
  - Addition.
  - Subtraction.
  - Multiplication.
  - Division.
  - Mixed operations.
- Score tracking.
- Pass/Fail result at the end of the game.
- Play again option.

## Technologies Used

- C++
- Procedural Programming
- Console Application

## How To Run

1. Clone the repository.
2. Open the source file in your preferred C++ IDE.
3. Compile and Run the project.
